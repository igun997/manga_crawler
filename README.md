## Manga Downloader 

### Requirement 

1. Imagick Extention 
2. PHP CLI >= 7.0 
3. GS (Ghost Script)

### Instalation 

1. Clone this repository 
2. enter into directory 
3. composer install 
4. enjoy

### How To Use 

``php manga <domain> <path>``

``php manga https://sektekomik.com https://sektekomik.com/manga/kaisar-shenwu/``

### Your Manga Path 
1. **download/[session]** (session based timestamp)
